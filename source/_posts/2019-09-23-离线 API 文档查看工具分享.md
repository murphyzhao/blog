写代码，API 用法记不住，那就需要一个程序员词典，推荐下两个好用的离线 API 查看工具，程序员的百科全书。

- [DASH](https://kapeli.com/dash)

仅可以用于 apple 产品，IOS 系统下。

- [zeal](https://zealdocs.org/)

zeal 就很良心了，Windows 下可用，文档也不比 DASH 少，界面干净，下载速度也比较快。
